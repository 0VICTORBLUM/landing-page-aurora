Estrutura da página

A landing page foi organizada em seções estratégicas usadas em páginas focadas em conversão:

Hero

Apresentação inicial da marca com destaque visual e chamada principal.

Sobre

Seção que apresenta o posicionamento da empresa e sua proposta no mercado de iluminação de luxo.

Portfólio

Galeria de projetos para reforçar autoridade e qualidade dos serviços.

FAQ

Perguntas frequentes com o objetivo de remover objeções comuns dos visitantes.

CTA

Seção de chamada para ação direcionando o usuário para contato via WhatsApp.

Tecnologias utilizadas

HTML5

CSS3

Media Queries

Essas tecnologias foram utilizadas para construir uma interface responsiva e organizada, mantendo foco em boas práticas de estruturação e layout.
